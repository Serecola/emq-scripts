# EMQ Scripts

Small collection of custom EMQ scripts

### [EMQ Chat Fix](https://github.com/hyther1111/emq-chat-fix/raw/main/emq-chat-fix.user.js) by [Hyther](https://github.com/hyther1111)
- Stops chat from auto-scrolling on new chat messages

### [EMQ Bulk Collections](https://github.com/Serecola/emq-scripts/raw/main/emq-chat-fix.user.js)
- Bulk add / delete items from a collection. Song IDs can be batch taken from [Bad EMQ Players](https://serecola.github.io/bad-emq-players/)

### [EMQ VN Shortcuts](https://github.com/Serecola/emq-scripts/raw/main/emq-vn-shortcuts.user.js)
- Displays shortcuts for VN titles in EMQ dropdown. Shortcut data is taken from [Bad EMQ Players](https://serecola.github.io/bad-emq-players/)
