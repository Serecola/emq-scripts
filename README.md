# EMQ Scripts

Small collection of custom EMQ scripts

### [EMQ Chat Fix](https://github.com/Serecola/emq-scripts/raw/main/emq-chat-fix.user.js)
- Stops chat from auto-scrolling on new chat messages

### [EMQ Bulk Collections](https://github.com/Serecola/emq-scripts/raw/main/emq-chat-fix.user.js)
- Bulk add / delete items from a collection. Song IDs can be taken from [Bad EMQ Players](https://serecola.github.io/bad-emq-players/)
